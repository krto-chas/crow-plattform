# CAF-003 Architecture Metamodel

## Purpose

Defines the common architectural object model used throughout CAF and CAS.

## Scope

The metamodel establishes common object types, mandatory attributes and relationships for architecture artefacts.

## Core Object Types

- Architecture
- Capability
- Service
- Process
- Information Asset
- Knowledge Asset
- Policy
- Standard
- Principle
- Decision
- Component
- Interface

## Mandatory Attributes

Identifier, Name, Description, Owner, Status, Version, Relationships and Lifecycle State shall be defined for every architecture object.

## Conformance

All CAF and CAS publications shall conform to this metamodel.

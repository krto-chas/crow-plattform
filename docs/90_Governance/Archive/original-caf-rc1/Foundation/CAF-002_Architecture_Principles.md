# CAF-002 Architecture Principles

## Purpose

Defines the governing architecture principles of CAF.

## Principles

1. Architecture before implementation.
2. Governance before automation.
3. Evidence before opinion.
4. Traceability by default.
5. Simplicity over unnecessary complexity.
6. Long-term sustainability.
7. Human accountability.
8. Consistent terminology.
9. Controlled evolution.
10. Reuse where practical.

## Application

All architectural decisions shall be evaluated against these principles.

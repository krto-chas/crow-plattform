# CAF-001 Core Concepts

## Purpose

Defines the core concepts used throughout the Crow Architecture Framework.

## Core Concepts

The following concepts are normative within CAF:

- Architecture
- Capability
- Service
- Governance
- Principle
- Standard
- Policy
- Information
- Knowledge
- Evidence
- Decision
- Component
- Interface
- Stakeholder
- Lifecycle
- Traceability

These concepts shall be used consistently across all CAF and CAS publications.

## Conformance

All CAF and CAS documents shall use the terminology defined by CAF-000 and this document.

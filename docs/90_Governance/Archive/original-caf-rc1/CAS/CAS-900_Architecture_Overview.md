# CAS-900 – Architecture Overview

## Purpose

This standard defines the purpose, scope and normative position of the Crow Architecture Standards (CAS) within the Crow Architecture Framework (CAF).

## Scope

CAS establishes the mandatory architecture standards that govern architecture artefacts, methods, governance and compliance across CAF implementations.

## Normative Status

Requirements in CAS are normative. Conformance shall be assessed against the requirements defined in applicable CAS documents.

## Standard Structure

Each CAS document shall include purpose, scope, normative requirements, rationale where appropriate, conformance criteria and references.

## Conformance

An implementation claiming conformance with CAF shall comply with all applicable mandatory requirements defined by CAS unless an approved governance exception exists.

## Summary

CAS provides the normative layer of CAF by translating architecture principles into verifiable requirements.
